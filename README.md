# Loved

Evolving Next.js application with a web page, status endpoint, Docker-based PostgreSQL, and an integration test.

**Category:** Full-Stack Web App

**Stack:** JavaScript, Next.js, React, PostgreSQL, Docker Compose, Jest

**Status:** Portfolio-ready educational project

## Run Locally

```bash
npm run dev
```
